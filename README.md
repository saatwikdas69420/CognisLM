# CognisLM
This is a small scale language model I'm currently working on, currently I have make the learning algorithm and even basic reasoning skills and the ability to learn from user data. There are four models avaliable, CognisLMv4 is the newest and most advanced, whilst CognisLMv3 is a bit more limited and CognisLMv2 and v1 are rudimentary and basic.
